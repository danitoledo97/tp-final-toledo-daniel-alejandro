Programación

La programación consiste en la construcción de un programa de código mediante un lenguaje, una sintaxis y un algoritmo finito y ordenado secuencialmente para llevar a cabo una tarea. A continuación nombraré las mas relevantes para las artes y el diseño multimediales.

Ramas mas artísticas

Programación Creativa

La programación creativa o creative coding es el uso del código como medio expresivo, donde el software funciona como herramienta artística y experimental.

  • Enfoque: Uso del código como medio artístico, expresivo o experimental.
  • Lenguajes/Tecnologías: Processing, p5.js, OpenFrameworks, TouchDesigner, Max/MSP, Unity, Arduino.
  • Aplicaciones: Instalaciones interactivas, arte digital, mapeo, visuales generativos, música algorítmica.
  • Claves: Interdisciplinariedad entre arte, diseño y tecnología

+CODE

"+CODE Cultura Digital impulsa y promueve la experimentación entre arte, programación creativa y nuevas tecnologías."

Programación en Vivo

También conocida como live coding forma parte de la creativa.

  • Enfoque: Generar música, visuales o performances en tiempo real escribiendo código frente al público.
  • Lenguajes/Tecnologías: Sonic Pi, SuperCollider, TidalCycles, Hydra.
  • Aplicaciones: Conciertos, jam sessions, performances audiovisuales.
  • Claves: Improvisación, inmediatez, mezcla de arte y tecnología en vivo.

Colectivo de Live Coders

"Un espacio de intercambio y producción colectiva para abordar, investigar y performar utilizando técnicas y herramientas de livecoding."

Artísticas según el uso

Programación Web

La programación web posibilita la creación de obras, proyectos y plataformas digitales accesibles desde navegadores, combinando imagen, sonido, texto, animación e interacción. Es una de las principales herramientas de la producción multimedia contemporánea.

  • Enfoque: Desarrollo de sitios y aplicaciones que funcionan en navegadores o servidores.
  • Lenguajes/Tecnologías:HTML, CSS, JavaScript, frameworks (React, Angular, Vue), backend (Node.js, PHP, Python, Java, etc.), bases de datos.
  • Aplicaciones: Páginas web, e-commerce, plataformas educativas, redes sociales, sistemas de gestión.
  • Claves: Usabilidad, seguridad, escalabilidad, interfaz y experiencia de usuario (UX/UI).

Programación de Videojuegos

En el desarrollo de videojuegos, existe una rama específica de la programación que se orienta a la creación de estos productos, utilizando principalmente lenguajes como C (antiguo y para proyectos indie), C++ y C# (derivados del anterior y actuales). En este proceso se toman en cuenta diversos factores como las mecánicas del juego, la física, el diseño de los personajes y los mundos, los efectos especiales (tanto sonoros como visuales), la música, la narración y la coherencia del sistema.

  • Enfoque: Creación de juegos digitales para distintas plataformas.
  • Lenguajes/Tecnologías: Motores como Unity (C#), Unreal Engine (C++/Blueprints), Godot (GDScript), frameworks 2D/3D.
  • Aplicaciones: Juegos de PC, consola, móvil, VR/AR.
  • Claves: Mecánicas de juego, narrativa, interactividad, optimización gráfica y de rendimiento.

Programación en Electrónica Creativa

La programación en electrónica creativa combina computación, electrónica y diseño para crear sistemas físicos interactivos con fines artísticos, expresivos o experimentales. Se vincula estrechamente con la computación física y las prácticas de arte tecnológico.

  • Enfoque: Control de hardware y dispositivos físicos mediante software.
  • Lenguajes/Tecnologías: C/C++ (Arduino, ESP32), Python (MicroPython, Raspberry Pi), VHDL/Verilog, MATLAB.
  • Aplicaciones: Robots, sensores, dispositivos IoT, domótica, prototipos electrónicos.
  • Claves: Integración hardware-software, tiempo real, eficiencia energética, interacción con el mundo físico.

Técnicas

Estas técnicas pueden combinarse entre sí: la programación visual puede ser no-code, low-code o híbrida, según el nivel de control y abstracción que ofrezca la herramienta.

Programación No-Code

  • Enfoque: Desarrollo de aplicaciones, sitios o flujos interactivos sin escribir código textual, mediante interfaces visuales, componentes predefinidos y reglas configurables.
  • Lenguajes/Tecnologías: Wix, Webflow, Bubble, Glide, Adalo, Notion, herramientas de automatización visual.
  • Aplicaciones: Sitios web, prototipos rápidos, landing pages, automatización de procesos, productos digitales simples.
  • Claves: Accesibilidad, rapidez de desarrollo, bajo umbral técnico, enfoque en diseño y experiencia de usuario.

Programación Low-Code

  • Enfoque: Desarrollo de software combinando interfaces visuales con la posibilidad de escribir código cuando se requiere mayor control o personalización.
  • Lenguajes/Tecnologías: Power Apps, OutSystems, Mendix, Node-RED, Unity Visual Scripting, Unreal Engine Blueprints, TouchDesigner.
  • Aplicaciones: Aplicaciones interactivas, sistemas personalizados, prototipos avanzados, experiencias multimedia complejas.
  • Claves: Equilibrio entre facilidad de uso y control técnico, escalabilidad, integración con código tradicional.

Programación Visual

  • Enfoque: Utiliza interfaces gráficas donde el código se construye arrastrando y conectando bloques o nodos en lugar de escribir texto.
  • Lenguajes/Tecnologías: Scratch (estándar educativo desde formación media), Blockly, Max/MSP, Pure Data, TouchDesigner, Node-RED, vvvv..
  • Aplicaciones: Educación y aprendizaje inicial de programación. Prototipado rápido de proyectos interactivos. Procesamiento de audio, video y datos en tiempo real.
  • Claves: Intuitiva, accesible, fomenta la creatividad y la experimentación sin necesidad de conocimientos avanzados de sintaxis.
